## 🏗️ 1. ARCHITECTURE: THE KDSC STACK
This repository is a **Knowledge-Driven Simulation Compiler (KDSC)**. This framework ensures that all world-behaviors are strictly derived from historical research and expressed through **Reusable Primitives**.

### **The 5-Layer Infrastructure**
1.  **RESEARCH**: Primary-source analysis (PDFs).
2.  **ONTOLOGY**: Unique entities and variants.
3.  **DESIGN**: Reusable Primitives (Scenes, Actors, Encounters, Resources).
4.  **COMPILE**: Deterministic building of the world-state.
5.  **RUNTIME**: Stable React/Zustand simulation.

---

## 🏛️ 2. THE THREE-PILLAR LINEAGE
We support three lineages of scholarly RPG play:
- **Ultima-like**: Reputation-gated systemic choices.
- **FF-like**: Authored dramatic scenes.
- **FTL-like**: Resource pressure and tactical crisis.
